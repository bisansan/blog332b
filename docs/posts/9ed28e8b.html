Django：URL规则re_path使用说明

发表于 2018-10-19 更新于 2021-04-21 分类于 Django

应用文件夹urls.py：

from django.urls import re_path
from . import views
urlpatterns = [
re_path(“^$”,views.app_one),
re_path(r”^list/(?P\d{3}).html”,views.app_three),
##对应的www.xxx.com/list/xxx.html
re_path(r”^book_(?P\d{3})_(?P\d{3})”,views.app_two)
##对应的www.xxx.com/book\_xxx\_xxx
]

应用文件夹views.py：

from django.http import HttpResponse
def app_one(request):
return HttpResponse(“APP首页”)

Create your views here.

def app_two(request,youinput,youe):
texta = “你的输入的是：%s”% youinput,youe
return HttpResponse(texta)

def app_three(request,youinput):
textb = “你的输入的是：%s”% youinput
return HttpResponse(textb)

能用path就尽量用path，除非要使用正则表达式才用re_path这样不会把简单的事情搞得复杂
