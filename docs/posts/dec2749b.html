路由之间跳转

this.$router.replace(path) => 跳转到指定url链接，但是它不会在浏览器中产生记录

this.$router.push(path) => 跳转到指定url链接，并在history栈中添加一条新的记录。

this.$router.go(val) => 在浏览器history记录中前进(正数)或者后退（负数）val步，当val为0时刷新当前页面。当返回的页数不存在就会报错
