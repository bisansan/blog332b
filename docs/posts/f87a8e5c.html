python 列表解析式、三元表达式和lambda

发表于 2019-09-10 更新于 2021-04-23 分类于 计算机编程

列表解析 List Comprehensions表达式：[expression for iter_val in iterable if cond_expr]

  • **[expression]**：最后执行的结果
  • **[for iter_val in iterable]**：这个可以是一个多层循环
  • **[if cond_expr]**：两个for间是不能有判断语句的，判断语句只能在最后；顺序不定，默认是左到右。

三元表达式 a if b else c

  • a : 真值
  • b : 判断条件，true时执行真值，false时执行假值
  • c : 假值

匿名函数 lambda x,y :x*3

前面为参数，后面是返回结果，以分号隔开，可以和三元表达式配合使用
